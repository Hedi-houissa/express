Description of express router :

Express is one of the popular web framework for Node.js. It provides wrapper to very useful functionality such as Rendering, Routing etc. In this tutorial i am going to cover Router() in Express framework. Express router is a class which helps us to create router handlers. By router handler i mean to not just providing routing to our app but also can extend this routing to handle validation, handle 404 or other errors etc.
